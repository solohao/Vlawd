# 桌面精灵表情原型（mascot-matrix）

独立可交互的桌面吉祥物表情视觉原型，**不接入生产代码**，仅用于设计验证。

## 运行

```bash
cd AI_cursor_v2/产品文档/概念设计图V2/mascot-matrix-prototype
python3 -m http.server 4189
# 浏览器打开 http://127.0.0.1:4189/
```

## 内容

- 六态（就绪/思考/执行/暂停/错误/完成）的 **6×6 状态过渡矩阵**：行=起始态、列=目标态，点格子播放过渡；对角线为该态待机动画。
- 五官轮廓从美术参考「表情图 A」忠实描摹并矢量化。
- **思考态**：眼形与逗号眼仁基于人工描摹的 右/上/左 三关键帧做逐帧 morph，并叠加"灵动"运动设计：
  - 扫视 + 凝视停顿（方向与时长随机，带回弹 overshoot）；
  - 凝视时的极轻微漂移（微扫视）；
  - 不规则眨眼（随机间隔 + 偶尔连眨）；
  - 呼吸起伏（轻微缩放/上浮/微摆）。
- 质感：果冻像素贴图填充（`jelly_tex.png`）+ 外发光，随矢量轮廓一起 morph。

## 文件

- `index.html` — 原型主体（SVG + 动画逻辑）。
- `clean_body.png` — 抹掉原就绪五官后的干净身体底图。
- `states_A.json` — 从表情图 A 描摹的六态五官轮廓。
- `thinking_frames.json` — 思考态 右/上/左 关键帧的眼形与眼仁轮廓（等点数、已对齐，供逐帧 morph）。
- `jelly_tex.png` — 果冻质感贴图。

## 说明

- 当前状态间过渡为交叉淡入（保真优先），非几何连续 morph。
- 若后续要更高质感/更丝滑，可评估 Lottie / Live2D，但工程量显著更高。
